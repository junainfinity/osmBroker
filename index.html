Local broadcasting

Expose the host Mac as an AI router.

Bind to localhost for private use, or to 0.0.0.0 when you want phones, laptops, or lab machines on the same network to call the broker.

Binding

Client configuration
curl http://192.168.1.18:8080/v1/models -H "Authorization: Bearer sk-osm-local-auth"

Endpoint emulation

/v1/models Returns the merged model catalog from enabled CLI providers.
/v1/chat/completions Accepts streaming inference requests and returns normalized SSE.
Bearer token required Rejects requests without the configured local API key.
